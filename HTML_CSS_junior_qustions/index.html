HTML/CSS

  • Селекторы:

    • * все элементы => слишком сильно грузит браузер.

    • #myId id="myId"

    • .myClass class="myClass"

    • div тег

    • div p дочерний тег

    • div > p только непосредственный дочерний тег (без учета вложенных)

    • a:hover псевдо класс

    • p::first-letter псевдо элемент

    • p+a p и сразу за ним a (строго)

    • p~a p и все идущие зa ним a (не строго)

    • input[type="text"]

    • a[href="http://only"] только эта ссылка

    • a[href*="http://only"]все ссылки этого сайта

    • a[href^="http"] все ссылки начинающиеся с ...

    • a[href$=".jpg"] все ссылки заканчивающиеся с ...

    • a[foo~="bar"] определенный атрибут из списка атрибутов

    • a:not(selector) все а исключая selector

    • li:nth-child(n) отсчет с 1

    • li:nth-last-child(n) отсчет с конца

    • ul:nth-of-type(n) отсчет с 1, элемент определенного типа

    • ul li:first-child первый дочерний элемент

    • li:last-child последний дочерний элемент

    • div p:only-child единственный дочерний элемент

    • li:only-of-type не имеет соседей в пределах родительского элемента

    • ul:first-of-type первый элемент заданного типа

  • Вес(специфичность) селекторов:

    •   ---- унаследованные стили

    •   0000 *

    •   0010 class

    •   0100 id

    •   1000 встроенные стили

    • 10000 !important
